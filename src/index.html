FAQ
Quels formats audio sont supportés?

L'application supporte tous les formats audio courants: MP3, WAV, OGG, FLAC, AAC, et plus encore.

Comment améliorer les performances?

Allez dans Paramètres > Performance et réduisez la qualité du rendu ou activez le mode économie d'énergie.

Mes fichiers sont-ils envoyés sur un serveur?

Non, tous les fichiers restent sur votre appareil. L'application fonctionne entièrement hors ligne.

Comment changer la couleur de visualisation?

Dans l'onglet Fichiers, utilisez le sélecteur "Couleur principale" pour personnaliser votre visualisation.

Puis-je utiliser mes propres arrière-plans?

Oui! Cliquez sur "Ajouter une image" dans la section Arrière-plans pour importer vos propres images.

L'export vidéo est-il disponible?

La fonctionnalité d'export vidéo est actuellement en développement et sera disponible prochainement.
